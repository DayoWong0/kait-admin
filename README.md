
# KAIT-Admin
A Free Vue.js + TailwindCSS Admin Dashboard Template.
![KAIT Dashboard](src/assets/images/screenshot.png?raw=true "KAIT Dashboard")

## Installation
```
git clone https://github.com/devoverid/kait-admin.git
cd kait-admin
yarn install
yarn serve
```
